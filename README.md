# Credit-Card-Financial-Dashboard-Using-Power-BI


## Project Overview
This project presents an interactive Power BI dashboard built to analyze credit card transaction data, customer demographics, and financial performance metrics. The dashboard delivers detailed insights into revenue trends, customer behavior, card usage, and regional contribution patternsâ€”supporting data-driven decision-making for financial institutions and business analysts.
